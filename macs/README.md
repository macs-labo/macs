## 本プロジェクトについて
携帯農薬検索実験室の農薬情報データベース（MacsDB）をダイレクトに検索し、その結果をスマホ等の小画面デバイスで見やすく表示するためのウェブアプリです。
ACFinderBE は小画面デバイスでは使いにくいので、こちらをお使いくください。
ACFinderBE とは逆に、検索から HTML レンダリングまで全てサーバ側で行うため、非力な SoC を搭載したタブレット等でも、MACS を使用した方が快適な場合があります。
